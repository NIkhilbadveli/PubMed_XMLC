These files were created as part of making the Parabel Windows binaries work on the dataset at hand.

convert_format.py ---> This file converts the given dataset into the Parabel input format by splitting into features and labels
process_data.py ---> Process the raw input documents and create the parabel data files.

parabel_predict.exe ---> Parabel prediction executable file. (Link to the original website where this can be downloaded)
parabel_train.exe ---> Parabel training executable file.

mlp_for_xmlc.py ---> Using a simple MLP for solving XMLC. Code taken from (Link here).
